description: Show local setup notes for the AC2 Codex plugin
---

Explain that the AC2 Codex plugin is configured through its Codex manifest
(`.codex-plugin/plugin.json`) and MCP registration (`.mcp.json`).

Tell the user to build the package with `npm run build`, install or load the
plugin in Codex, then call `ac2_pair` to connect their wallet. Do not edit
Claude settings files for this plugin.
